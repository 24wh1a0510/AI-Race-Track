AI Race Track is an interactive Streamlit application that compares multiple Large Language Models (LLMs) side-by-side in real time. The same prompt is sent to multiple AI models through OpenRouter, allowing users to evaluate response quality, latency, token usage, and estimated cost in a single dashboard.
The project was inspired by a Multi-Model Comparison Tool and enhanced with a custom race-track themed interface to make AI model evaluation more engaging and intuitive.

## рџ”’ Security Notes
The following files are intentionally excluded from GitHub:
```text
.env
.venv/
__pycache__/
*.pyc
```

Reasons:
* `.env` contains sensitive API credentials.
* `.venv` contains local project dependencies.
* `__pycache__` and `.pyc` files are automatically generated by Python.
Never commit API keys or secrets to a public repository.
